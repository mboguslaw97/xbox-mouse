Description:
This is a program allows the use of a gamepad as a mouse. By default, it is set up to work with an xbox controller,
but can be easily modified to work with other devices. To see the button encodings of an alternative device, set DISPLAY = True.


Button Mappings:
 - A -> Left mouse click
 - B -> Right mouse click
 - D-Pad vertical -> mouse sensitivity
 - D-Pad horizontal -> scroll sensitivity
 - Select -> Open on screen keyboard (Buggy)
 - Start -> Stop running xbox_mouse.py


Dependencies:
 - inputs
 - pyautogui


Bugs:
 - Unable to click on the on screen keyboard
